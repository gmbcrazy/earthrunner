EarthRunner

Run anywhere on Earth from a treadmill

Interactive navigation through real places

Contact: contact@earthrunner.org

EarthRunner is an experimental immersive running system that transforms treadmill exercise into a journey through real cities, landmarks, and landscapes.

Run through real places

Move through cities, landmarks, and landscapes as if each workout were a new destination.

Turn and explore interactively

Change direction naturally and navigate real environments instead of following a fixed route.

AI-powered audio guidance

Hear context-aware narration that adds local detail and makes every route feel more alive.

Potential applications

  • Fitness and immersive exercise
  • Rehabilitation and recovery
  • Education and cultural exploration
  • Public exhibitions and interactive installations
Built independently as a personal project using Unreal Engine and Cesium.
